設定されている場合、不安定なビルド結果として解釈されるバッチエラーレベルの結果。 最終エラーレベルが値と一致する場合、ビルド結果は不安定に設定され、次のステップが続行されます。 サポートされている値は、Windowsのようなシステムの最も広いエラーレベル範囲に一致します。 Windows NT4以降では、ERRORLEVELは4バイトの符号付き整数として保存され、最大値と最小値は それぞれ2147483647と-2147483648になります。 システムのようなDOSは1バイトを使用し、0〜255のエラーレベルを生成します。 値0は無視され、デフォルトの動作の一貫性を保つためにビルドが不安定になることはありません
