このリストには、構成済みの更新サイトによって発行され、現在インストールされているコンポーネントに関連するすべての警告が含まれています。 これらは通常、セキュリティに関連しています。 公開されているが、現在インストールされているコンポーネントに関連しない警告（影響を受けるコンポーネントがインストールされていないか、影響を受けていないバージョンがインストールされているため）は、ここには表示されません

チェックしたエントリ（デフォルト）はアクティブです。つまり、管理モニターで管理者に表示されます。 エントリのチェックを外して非表示にすることができます。 これは、特定の警告を評価し、それが環境または構成に適用されないことを確信していて、指定されたコンポーネントの継続使用がセキュリティ上の問題を構成しない場合に役立ちます。

特定の警告のみを無効にできることに注意してください。特定のコンポーネントに関するすべての警告を無効にすることはできません。 警告の表示を完全に無効にする場合は、システムの構成で管理モニターを無効にできます。
